Health Challenge Tracker

Features
Add Workout
Filter Workouts
View Progress
Pagination

### Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/raghavamamidi/health_tracker.git
   cd health_tracker
   ```

2. Install dependencies

```bash
   npm install
```

3. Run the project

```bash
   npm start
```

4. You can access the project at http://localhost:4200
